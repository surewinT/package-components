# package-components 

本项目对基础组件进行封装，以便更贴合实际业务。

# 目录说明
项目工程基于vue-admin-template搭建，项目工程目录可以查看固网

```
├── components            # 业务组件库（封装的组件都放在这里）
│   │── layout-components # 项目的主题组件（vue-admin-template官方的）
│   │── p-el-checkbox     # 改造的el-checkbox
├── views                 # 组件案例
│   │── login             # 登录页
│   │── p-el-checkbox     # 改造的el-checkbox案例
```
